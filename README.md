# MKWS26 Cantera H2 Combustor

Cantera-based parametric study of hydrogen-enriched methane-air combustion under simplified aircraft combustor conditions.

## Planned analyses

- adiabatic flame temperature,
- equilibrium product composition,
- ignition delay,
- influence of equivalence ratio,
- influence of hydrogen fraction,
- influence of initial temperature and pressure,
- simplified combustor residence-time analysis.

## Tools

- Python
- Cantera
- NumPy
- pandas
- Matplotlib
